# Formulario Matriz CONDAPAV

Formulario estatico para preenchimento da Matriz Exigencia x Resposta do projeto CONDAPAV/MMA.

## Publicacao no GitHub Pages

Este repositorio foi preparado para publicacao estatica diretamente pelo GitHub Pages.

Arquivos publicados:

- `index.html`
- `formulario_matriz.html`
- `formulario_matriz.css`
- `formulario_matriz.js`

O formulario salva respostas no `localStorage` do navegador e permite exportar/importar JSON.
